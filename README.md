# Portfolio — Creative Computing (Level 6)

This is a minimal, dark-themed static portfolio scaffold. It’s designed to be easy to update with YouTube project links, photography, and resume.


To preview locally, run:
```powershell
python -m http.server 8000
```
Then open http://localhost:8000
