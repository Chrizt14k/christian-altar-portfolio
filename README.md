# Portfolio — Creative Computing (Level 6)

This is a minimal, dark-themed static portfolio scaffold. It’s designed to be easy to update with your YouTube project links, photography, and resume.

How to use
- Replace `Resume.pdf` in the project root with your actual PDF resume.
- Replace project cards in `index.html` with your titles, descriptions, and links.
- Put photography files in an `assets/` folder and update the photo grid markup.

To preview locally, run:
```powershell
python -m http.server 8000
```
Then open http://localhost:8000

Deployment
- Use GitHub Pages: set source to `main` branch and `/root`.
- Or deploy to Netlify / Vercel by connecting the repo.
